========================================
  FOR STUDENTS
========================================

You can follow along with:
- The PDF on screen (recommended)
- Your physical book using page/line numbers
- Both at the same time

Use the speed controls:
- 0.75x = Slower (great for EL learners)
- 1.0x = Normal speed
- 1.25x = Slightly faster
- 1.5x = Much faster
